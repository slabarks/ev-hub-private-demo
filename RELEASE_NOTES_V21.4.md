# V21.4 release notes

## Portfolio Financial Performance board review

- Added **Manage portfolio terms** beside the main site-performance heading. It combines bulk landlord controls with portfolio-wide funding controls.
- Added bulk actions to apply eligible confirmed funding, remove applied funding, and reset all funding choices to database values. Review-required and funding-over-CAPEX anomalies are clearly identified.
- Funding inputs now accept exact whole-euro values.
- Reversed CAPEX variance to the board convention: actual minus model. Overspend is positive/red; underspend is negative/green.
- Made **Site financial performance** a prominent investor section heading.
- Stored fallback now displays the actual snapshot date (13 Jul 2026).
- Added actual performance beside every Next 12m kWh forecast and placed the forecast-vs-actual percentage directly beside the forecast.
- Changed the primary model-performance comparison to historical actual delivered kWh versus the current calibrated age-matched model over the exact same elapsed period. The forward benchmark remains visible in the forecast audit.
- Added MIC and the capacity-rate calculation to the Energy & network cell.
- Retained the existing rolling-30 forecast graph and the conservative phased 1% traffic-growth assumption.
